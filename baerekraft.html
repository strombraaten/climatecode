Bærekraftig Webdesign

Våre bærekraftige valg

  • Tailwind CSS: Vi bruker Tailwind CSS for effektiv og optimalisert styling. Dette reduserer filstørrelsen og forbedrer lastetiden ved å kun inkludere de stilene som faktisk brukes på nettsiden.
  • Minimal JavaScript: Vi har bevisst valgt å minimere bruken av JavaScript. Den eneste JavaScript-koden vi bruker er for å håndtere dark mode-funksjonaliteten, som forbedrer brukeropplevelsen og potensielt sparer energi på enheter med OLED-skjermer.
  • Systemskrifttyper: Vi bruker systemskrifttyper som er innebygd i alle nettlesere. Dette eliminerer behovet for å laste ned ekstra skrifttyper, noe som reduserer dataoverføringen og forbedrer lastetiden.
  • Enkel HTML og CSS: Ved å holde oss til enkel HTML og CSS, reduserer vi kompleksiteten og filstørrelsen til nettsiden. Dette resulterer i raskere lasting og mindre energiforbruk både på serveren og klientens enhet.
  • Responsivt design: Nettsiden er designet for å være responsiv og fungere godt på alle enheter. Dette eliminerer behovet for separate mobilsider og reduserer den totale datamengden som må overføres og vedlikeholdes.
  • Dark mode: Implementeringen av dark mode gir ikke bare en bedre brukeropplevelse, men kan også bidra til å redusere energiforbruket på enheter med OLED-skjermer.

Kontinuerlig forbedring

Vi er forpliktet til å kontinuerlig forbedre bærekraften til vår nettside. Vi følger nøye med på nye teknologier og beste praksis innen bærekraftig webdesign, og vil implementere ytterligere forbedringer etter hvert som de blir tilgjengelige.
