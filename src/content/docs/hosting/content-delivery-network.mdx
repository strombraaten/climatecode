Content Delivery Network, eller CDN som det gjerne kalles,

import { Aside } from '@astrojs/starlight/components';

<Aside type="note" title="Forklar det som om jeg var 5 år gammel">
  Du kan se på en bokhandel som et Content Delivery Network. Du kan kjøpe samme
  bok i små og store byer over hele verden, og være sikker på at du får nøyaktig
  det samme innholdet i boka. På samme måte kan brukere få tilgang til innholdet
  på nettsiden din fra servere nær dem. Noe som gjør selve leveringen raskere og
  mer pålitelig.
</Aside>
