# Om Klimakode

Klimakode er et initiativ for å dokumentere og dele konkrete grep vi kan ta for å lage mer bærekraftige digitale tjenester.

## Formål

Målet med prosjektet er todelt:

1. **Utforske beste praksis** for hvordan du kan redusere karbonavtrykket til tjenestene du bygger
2. **Dele kunnskap** som kan hjelpe produktteam med å ta mer miljøvennlige valg

## Bidra til prosjektet

Klimakode er et åpent prosjekt som _du_ kan være med å skape. Det er flere måter du kan bidra på:

- **Del kunnskapen** - Tips andre om prosjektet og spre budskapet
- **Foreslå endringer** - Lag en issue hvis du ser noe feil (trykk på "Rediger denne siden" nederst til venstre)
- **Del erfaringer** - Send inn pull requests med dine egne anbefalinger

## Kontakt

Prosjektet vedlikeholdes på [GitHub](https://github.com/strombraaten/climatecode), hvor du kan følge utviklingen, rapportere problemer eller bidra direkte til kodebasen.
